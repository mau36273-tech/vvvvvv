# GitHub-safe upload notes

This archive intentionally excludes `ios/App/App/public/` because it is generated by Capacitor and duplicates the `web/` folder.

After cloning/downloading on a Mac, regenerate it with:

```bash
npm install
npm run build:web
npx cap sync ios
cd ios/App
pod install
open App.xcworkspace
```

Open `App.xcworkspace`, not `App.xcodeproj`.
